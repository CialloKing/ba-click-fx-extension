Unofficial fan-made extension

Click rings, particles, and cursor trails for the web.

BA Click FX adds configurable, blue-themed visual feedback to ordinary webpages. It runs locally with Canvas 2D and includes per-site controls, color settings, and performance modes.

View releases Installation & support

Single purpose

Display configurable pointer click animations and cursor trails on ordinary webpages.

Local rendering

No ads, analytics, telemetry, developer server, or remotely hosted code.

Transparent controls

Effects can be disabled globally or per website, and preferences can be reset at any time.

Data disclosure

Pointer coordinates are processed temporarily in memory to draw effects. The current website origin is processed locally for the per-site switch; an origin is saved only when the user explicitly disables that website. Visual and site preferences are stored through the browser's sync storage. The developer receives none of this information.
