Privacy Policy for Microsoft Edge

BA Click FX has no developer-operated server and makes no network requests. The developer cannot access pointer data, browsing activity, or saved Microsoft Edge preferences.

Information processed in Microsoft Edge

The extension processes pointer coordinates temporarily in memory to draw click effects and cursor trails. It processes the current webpage origin locally to apply the per-site switch. If the user explicitly disables a website, only that origin is saved as a site-specific preference.

Enabled states, color, opacity, scale, quality, and site-specific rules are saved using the Microsoft Edge implementation of the browser storage.sync API. The extension does not read webpage text, forms, passwords, cookies, communications, account information, or payment information.

Use and Microsoft Edge sync

Information is used solely to render visible pointer effects and apply user-selected settings. When Microsoft Edge sync is enabled, Microsoft may synchronize extension preferences through the user's Microsoft account according to the user's Edge sync controls and Microsoft's privacy statement. The extension developer does not operate that synchronization and cannot access the synchronized values.

Sharing, retention, and controls

The developer does not sell, rent, share, or transfer user information. Pointer coordinates are not retained. Saved settings remain until changed or reset, extension data is cleared, Edge sync is disabled, or the extension is uninstalled. The extension contains no advertising, analytics, telemetry, profiling, or remote code.

Changes and contact

If the Microsoft Edge version's data practices change, this policy and the Partner Center disclosures will be updated before release. Questions can be submitted through the project issue tracker.

Microsoft Edge 隐私政策

BA Click FX 没有开发者服务器，也不会发起网络请求。开发者无法访问鼠标数据、浏览活动或 Microsoft Edge 中保存的设置。

在 Microsoft Edge 中处理的信息

插件只在内存中临时处理鼠标坐标，用于绘制点击特效和光标拖尾；当前网页 origin 只在本地用于站点开关。只有当用户主动关闭某个网站时，该 origin 才会保存为站点规则。

全局开关、颜色、不透明度、大小、画质和站点规则通过 Microsoft Edge 实现的浏览器 storage.sync API 保存。插件不会读取网页正文、表单、密码、Cookie、通信内容、账号信息或支付信息。

使用与 Microsoft Edge 同步

上述信息只用于渲染可见特效和应用用户选择的设置。如果用户启用了 Microsoft Edge 同步，Microsoft 可能根据用户的 Edge 同步设置和 Microsoft 隐私声明，通过用户的 Microsoft 账号同步扩展设置。开发者不运营该同步过程，也无法访问同步内容。

共享、保留与控制

开发者不会出售、出租、分享或转移用户信息。鼠标坐标不会保留；保存的设置会一直存在，直到用户修改或恢复默认设置、清除扩展数据、关闭 Edge 同步或卸载插件。插件不包含广告、分析、遥测、画像或远程代码。

变更与联系

如果 Microsoft Edge 版本的数据处理方式发生变化，本政策和 Partner Center 披露会在发布前更新。问题可通过项目的 公开 Issue 页面提交。
