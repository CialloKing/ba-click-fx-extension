Privacy Policy

BA Click FX has no developer-operated server and makes no network requests. The developer cannot access pointer data, browsing activity, or saved preferences.

Information processed

Pointer coordinates and pointer events are processed temporarily in memory to draw click rings, particles, and cursor trails. They are not logged or retained. The current webpage origin is processed locally to apply the per-site switch. If the user explicitly disables a website, that origin is saved as a site-specific preference.

Visual preferences—including enabled states, color, opacity, scale, quality, and site-specific rules—are saved with the browser-provided storage.sync API. The extension does not read webpage text, form values, passwords, cookies, personal communications, account information, or payment information.

Use, storage, and transmission

Information is used only to render visible effects, apply appearance and performance choices, and remember site-specific enable states. The browser may synchronize preferences through the user's signed-in browser account when sync is enabled. That process is operated by the browser vendor and governed by the user's browser settings and the vendor's privacy policy.

Sharing, sale, and advertising

The developer does not sell, rent, share, or transfer user information. BA Click FX contains no advertising, analytics, telemetry, profiling, or remotely hosted code.

Retention and controls

Pointer coordinates exist only while an effect is rendered. Preferences remain in extension storage until changed or reset, extension data is cleared, browser sync is disabled, or the extension is uninstalled. A saved site rule can be removed by enabling that site again or restoring defaults.

Limited Use

The use of information received from browser APIs adheres to the Chrome Web Store User Data Policy, including the Limited Use requirements. Information is used only for the extension's single, user-facing purpose.

Changes and contact

If data practices change, this policy and store disclosures will be updated before release. Questions can be submitted through the public issue tracker.

隐私政策

BA Click FX 没有开发者服务器，也不会发起网络请求。开发者无法访问鼠标数据、浏览活动或保存的设置。

处理的信息

插件只在内存中临时处理鼠标坐标和指针事件，用于绘制点击圆环、粒子和光标拖尾，不会记录或保留这些坐标。当前网页的 origin 只在本地用于应用站点开关；只有当用户主动关闭某个网站时，该 origin 才会保存为站点规则。

全局开关、颜色、不透明度、大小、画质和站点规则通过浏览器提供的 storage.sync API 保存。插件不会读取网页正文、表单值、密码、Cookie、通信内容、账号信息或支付信息。

使用、存储与同步

上述信息仅用于绘制用户可见的特效、应用外观与性能选择，以及记住站点开关。如果用户启用了浏览器同步，浏览器厂商可能通过用户登录的浏览器账号同步设置；该过程受用户的浏览器设置和厂商隐私政策约束。

共享、出售与广告

开发者不会出售、出租、分享或转移用户信息。插件不包含广告、分析、遥测、画像或远程代码。

保留与用户控制

鼠标坐标只在特效渲染期间存在。用户可通过修改或恢复默认设置、清除扩展数据、关闭浏览器同步或卸载插件来删除保存的信息；重新启用网站即可删除对应的站点规则。

变更与联系

如果数据处理方式发生变化，本政策和商店披露会在新版本发布前更新。问题可通过项目的 公开 Issue 页面提交。
