Installation and troubleshooting

Getting started

  1. Install BA Click FX from a supported browser store or load the unpacked dist directory in developer mode.
  2. Refresh webpages that were already open when the extension was installed.
  3. Click or move the pointer on an ordinary HTTP or HTTPS webpage.
  4. Open the toolbar popup to adjust effects or disable the current website.

Where effects cannot run

Browsers block content scripts on internal pages, browser extension stores, and some built-in viewers. The extension intentionally runs only in top-level documents, so pointer events inside cross-origin iframes may not show effects. Local file:// pages require the user to enable file access in the extension's browser settings.

Resetting settings

Open the popup and select Reset. To remove all extension data, uninstall the extension or clear its stored data through the browser's extension management interface.

Report a problem

Open a GitHub issue with the browser version, extension version, affected website type, reproduction steps, and any error shown on the extension management page. Do not include passwords, private browsing data, or other sensitive information.
